# Conceitos Básicos de CSS

Esta pasta contém arquivos que demonstram os fundamentos da linguagem **CSS (Cascading Style Sheets)**, utilizada para estilizar e dar layout a documentos HTML.

## Funcionalidades e Arquivos

| Conceito | Arquivos de Exemplo | Descrição |
| :--- | :--- | :--- |
| **Estilos Inline, Interno e Externo** | `inline-style.html`, `internal_style.html`, `pagina1.html`, `pagina2.html`, `estilo.css` | Demonstra as três formas principais de aplicar CSS a um documento HTML: diretamente no elemento, dentro da tag `<style>` e através de um arquivo externo. |
| **Seletores de Classes e IDs** | `classes-ids.html` | Exemplos de como usar seletores de classe (`.nome-classe`) e seletores de ID (`#nome-id`) para aplicar estilos específicos. |
| **Estilos de Texto** | `estilos-texto.html`, `fonte-cores.html`, `tamanho-textos.html` | Demonstra propriedades CSS para formatação de texto, como `font-family`, `font-size`, `color`, `text-align`, `text-decoration`, entre outros. |
| **Cores e Imagens de Fundo** | `cor-imagem-fundo.html` | Exemplos de como definir cores de fundo (`background-color`) e usar imagens como fundo (`background-image`). |
| **Bordas** | `bordas.html` | Demonstra as propriedades para estilizar bordas de elementos, como `border-style`, `border-width` e `border-color`. |
| **Elementos `div` e `span`** | `div-span.html` | Explica a diferença entre os elementos de bloco (`<div>`) e elementos inline (`<span>`) e como eles são usados para agrupar e estilizar conteúdo. |

## Como Usar

Para visualizar o resultado de qualquer um desses arquivos, abra o arquivo HTML correspondente em seu navegador. O arquivo `estilo.css` é o arquivo de estilo externo utilizado por alguns dos exemplos. Os arquivos são focados em um único conceito de CSS.
