# Conceitos Básicos de HTML

Esta pasta contém arquivos que demonstram os fundamentos da linguagem **HTML (HyperText Markup Language)**, essenciais para a estruturação de qualquer página web.

## Funcionalidades e Arquivos

| Conceito | Arquivos de Exemplo | Descrição |
| :--- | :--- | :--- |
| **Caracteres Especiais** | `caracteresEspeciais.html` | Demonstra o uso de entidades HTML para exibir caracteres especiais (como &amp;, &lt;, &gt;) que não podem ser digitados diretamente ou que têm significado reservado. |
| **Inserção de Imagens** | `imagem.html` | Exemplo de como incorporar imagens em uma página web usando a tag `<img>` e seus atributos essenciais (`src`, `alt`). |
| **Formulários** | `formularios.html` | Contém a estrutura básica de um formulário HTML, incluindo diferentes tipos de campos de entrada (`<input>`), `textarea`, `select` e botões de submissão. |
| **Navegação e Links** | `contato.html`, `index.html` | Demonstra a criação de hiperlinks (`<a>`) para navegação entre páginas e recursos externos, utilizando caminhos relativos e absolutos. |
| **Tabelas** | `tabelas.html` | Exemplo de como estruturar dados em linhas e colunas usando as tags `<table>`, `<tr>`, `<th>` e `<td>`. |

## Como Usar

Para visualizar o resultado de qualquer um desses arquivos, basta abri-los diretamente em seu navegador web. Eles são exemplos simples e focados em um único conceito.
