# Projetos do Curso de Desenvolvimento Web

Esta pasta contém diversos projetos práticos desenvolvidos durante o curso, que aplicam e combinam os conceitos de HTML e CSS.

## Lista de Projetos

| Projeto | Descrição | Foco Principal | Pasta |
| :--- | :--- | :--- | :--- |
| **Ana Bella - Modelo de Página** | Criação de um site simples para uma modelo, com múltiplas páginas (biografia, campanhas, contato). | Estrutura HTML, links, imagens, layout básico. | `projetoAnaBella` |
| **CSS Intermediário** | Conjunto de exemplos que exploram conceitos avançados de CSS. | Modelo de caixa, elementos flutuantes (`float`), posicionamento, sobreposição (`z-index`), `inline-block`. | `projetoCssIntermediario` |
| **Layouts Avançados (CSS3)** | Exemplos de layouts e componentes mais complexos. | Estilização de formulários, tabelas, navegação horizontal/vertical, abas, layout fixo. | `css3-intermediario-avancado` |
| **TecBlog** | Um projeto de blog com layout de duas colunas e estilização de posts. | Layout de múltiplas colunas, estilização de listas e links, organização de conteúdo. | `projetoTecBlog` |
| **Site de Notícias** | Simulação de um portal de notícias com diferentes seções. | Layout complexo, uso de `div`s para estruturação, navegação e estilização de conteúdo jornalístico. | `projeto_site_noticias` |
| **Chalé Hotel** | Projeto de um site para um hotel, com foco em design e layout. | Layout elegante, uso de imagens de fundo, estilização de menus e áreas de conteúdo. | `projetochalehotel` |
| **Projeto Final (Modelo)** | Um projeto de site mais completo, servindo como modelo final. | Integração de múltiplos conceitos, estrutura de navegação e conteúdo mais robusta. | `projetoFinal` |

## Documentação Detalhada

Para cada projeto, você pode encontrar arquivos HTML e CSS que demonstram as técnicas utilizadas.

### Navegação e Layouts (`css3-intermediario-avancado`)

Esta subpasta é especialmente rica em exemplos de layout e navegação:

*   **Navegação Horizontal e Vertical:** Demonstra como transformar listas HTML em menus de navegação.
*   **Estilização de Formulários e Tabelas:** Foca em tornar elementos padrão mais atraentes e usáveis.
*   **Layout Fixo:** Exemplo de um layout que mantém a largura fixa, independentemente do tamanho da tela.

Recomenda-se explorar os arquivos `index.html` dentro de cada subpasta para ver os exemplos em ação.
